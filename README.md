# Enterprise Compliance Management System

A full-stack **Enterprise Compliance Management System** built with  
**Node.js (TypeScript)**, **Angular**, and **MySQL (XAMPP)**.

This application helps organizations manage compliance disclosures, audits,
and user access in a secure and scalable way.
